Programing Languages

Being able to program a computer is a fun and useful skill to have. In this post, we will explore some of the most popular languages along with reasons to learn them!

All About Languages

We all know that learning to speak a new language is a great idea - it keeps our brains healthy and helps us to see the world in new ways. But did you know that learning computer programing languages is also a smart idea? This post will hopefully help you understand a little more about why this is so!

Lots of people immediately think that the main reason to learn how to program a computer is because there are lots of jobs which require that knowledge. Whilst this is one reason to learn programing, I argue that this is less important than many of the other reasons!

Learning to program is fun because you have to apply knowledge to solving problems. Furthermore, it makes you smarter just like learning other "human" languages makes you smarter. In addition, learning to program helps you see the world differently - the skills you develop whilst programing will impact on how you approach life in general.

Perhaps most importantly, learning to program makes you powerful! It gives you the power to positively change your own life and the lives of others by giving you the ability to creatively solve problems to real world problems!

"Working with the computer gives rise to many opportunities to transcend asocial behaviour, because it produces exciting and visually interesting things to share, whether it's creating video games, computer art or sharing exciting Web sites." - Seymour Papert

Common Programming Languages

Python

Python is a very popular language which was created by Guido van Rossum and released in 1991. It is relatively easy to learn and is used in lots of different areas ranging from cyber security to big data projects with server side web development found along the way!

Ruby

Ruby is another popular language which was created by Yukihiro Matsumoto and released in 1995. Right from the start, it has been developed with ease of use in mind. Ruby is a stand alone language which can be used for many purposes, but lots of people associate it with Ruby on Rails which is a great web development framework built on Ruby.

C++

C++ is another hugely popular language which was built by Bjarne Stroustrup as an extension to the C programming language. C++ gives developers more control over system resources and memory. It is closely related to C, but C++ includes the use of classes and objects since it is an object-oriented programing language.

JavaScript

This is an interesting language since there are different implementations of it. For some time, JavaScript was confined to running only in web browsers - and it still excels as a language to use in front-end web app development - but in 2009 Ryan Dahl developed Node JS which broke JavaScript free and means it can now be used to develop the back-end of websites, too!

Which One to Learn?

In many ways it doesn't really matter! Every language is useful and will bring you great benefits. It makes sense to start with one you feel comfortable with and then start to learn additional languages as you go. Just like when learning to speak different languages, once you know one, it is easier to pick up others!
